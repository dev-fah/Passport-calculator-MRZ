# Passport-calculator-MRZ
Le **Passport Calculator MRZ** est un outil qui lit, décode et vérifie les informations contenues dans la zone de lecture optique (MRZ) d’un passeport, comme le nom, numéro, date de naissance et expiration. Il contrôle l’intégrité via les checksums et peut parfois générer une MRZ pour tests ou applications légitimes.
